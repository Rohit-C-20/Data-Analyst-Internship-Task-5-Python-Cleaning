# Task 5 - Python Basics (Pandas Data Cleaning)

## Dataset
Titanic Dataset

## Objective
Performed basic data cleaning using pandas.

## Steps Performed
- Loaded dataset using pandas
- Checked dataset structure using head() and info()
- Found missing values using isnull().sum()
- Filled missing Age values using median
- Dropped Cabin column due to high missing values
- Removed duplicates
- Converted datatypes
- Created new feature: FamilySize
- Saved cleaned dataset as cleaned_data.csv
